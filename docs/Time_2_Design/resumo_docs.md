# Resumo das Documentações Produzidas pelo Time 2

## Tutorial de Uso

O tutorial de uso do sistema encontra-se no arquivo `README.md`, localizado na pasta `adk/`. Ele apresenta um guia passo a passo detalhando como utilizar o sistema.

## Prompts de Teste

Disponibilizamos na pasta `adk/` o arquivo `prompt-test.md`. Este documento contém uma série de prompts de teste com seus respectivos resultados esperados, com o objetivo de facilitar a validação e o teste do sistema.

## Diagrama de Fluxo

O arquivo `AI4SE - Fluxo HU(s) ótimas Design.pdf` ilustra o fluxo de funcionamento do sistema para Histórias de Usuário (HUs) consideradas ótimas (aquelas que não resultam na geração do `Doubt_Artifact.md`). O diagrama apresenta uma visão geral da operação do sistema e de suas restrições.
